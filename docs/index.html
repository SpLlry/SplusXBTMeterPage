核心功能亮点

实时监控

自动扫描并监控已连接的蓝牙设备电量状态，支持 Windows 11/10 系统托盘显示。

低电量弹窗提醒

告别设备突然断电，低电量自动弹窗提醒，无需手动查看设备管理。
